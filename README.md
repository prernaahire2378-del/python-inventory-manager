# Python Inventory Manager

This project is a simple Python-based inventory management system that reads and updates book data stored in a JSON file. 
The program loads the inventory, appends new book entries, and displays a formatted report of all books.

## Features
- Read data from JSON file
- Add new book entries
- Display formatted inventory report
- Demonstrates Python file handling and JSON module

## Files in Project
inventory.json – stores book inventory data  
inventory_manager.py – main Python program  

## How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

python inventory_manager.py
